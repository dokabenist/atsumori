熱盛と出てしまうChrome拡張機能＆ブックマークレット

Chrome拡張機能

  • Chromeウェブストアからインストールして下さい
  • インストール後は特に何もしなくても自動で有効になります

ブックマークレット

  • 「熱盛」 ←このアンカーを右クリックして「ブックマークに追加」するか、アンカー上でドラッグしてブックマークバーにドロップなどして、ブックマーク登録して下さい
  • 目的のページを開いたら、登録したブックマークを選択して下さい（手動でブックマークを選択しないと有効になりません）
  • Chrome以外の方でも使えます
    • Chrome、Firefoxで動作を確認
    • Safariでは動作しないようです…
